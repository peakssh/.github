# peakssh

Open-source SSH management suite.
Built for speed, security, and full transparency.

[peakssh.dev](https://peakssh.dev) | [contact](https://peakssh.dev/contact) | [documentation](https://peakssh.dev/docs)

---

### Key Information
* **License:** MIT (Fully open-source).
* **Platforms:** Desktop (Linux, macOS, Windows), Mobile (iOS, Android) and [webapp](https://app.peakssh.dev/).
* **Control:** A lightweight alternative for managing your SSH connections.

### Synchronization
Cross-client synchronization is handled via the optional **peakssh sync plugin**.
* Features **End-to-End Encryption (E2EE)**.
* Full peakssh configuration synced across clients.
